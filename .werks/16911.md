# HW/SW inventory subtree hardware/nwadapter not showing IP addresses anymore

key        | value
---------- | ---
date       | 2026-03-04T12:08:14+00:00
version    | 3.0.0b1
class      | fix
edition    | community
component  | checks

With [werk 18704](https://checkmk.com/werk/18704) there is a dedicated approach for IP addresses to go to `networking/addresses`,
making those in `hardware/nwadapter` redundant.

This change removes IP addresses from `hardware/nwadapter` - network interface adapters can still be found here.
