# host_labels_if: don't filter IPv4 APIPA (link-local) addresses

---------- | ---

version    | 2.5.0b2

edition    | community

APIPA addresses (169.254.0.0/16) are link-local for IPv4 and had been filtered when concidering whether a host is 'singlehomed' or 'multihomed',
but represent legitimate network configurations.

This change limits fitering for link-local to IPv6.
