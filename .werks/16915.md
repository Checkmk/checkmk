# inventory_win_networkadapter: dropped parsed data leads to empty HW/SW inventory

key        | value
---------- | ---
date       | 2026-04-01T13:48:23+00:00
version    | 2.6.0b1
class      | fix
edition    | community
component  | checks
level      | 1
compatible | yes

`parse_win_networkadapter()` used for windows based network topology features like host labels and network adapter and IP address inventory had been turned into a (single use) generator, resulting data not being propagated to the HW/SW inventory.

This change makes `parse_win_networkadapter()` return a list and thus feeding the inventory functions again.
