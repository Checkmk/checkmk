# Add view specific row limit

---------- | ---

component  | multisite

This change adds a new configuration parameter "Row limit", which can
be used to set the maximum number of rows shown for a view to a fixed
number. This option can be useful when creating top N lists.

The new parameter overrides the global soft and hard row limit behaviour for
each view which has this option configured.
