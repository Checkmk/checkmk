# Close dropdown when clicking currently active option

key        | value
---------- | ---
date       | 2025-11-24T15:26:20+00:00
version    | 2.4.0p17
class      | fix
edition    | cre
component  | wato

The dropdowns in our updated form rendering engine (used in Quick Setup, DCD and other pages)
would stay open when clicking the currently active option.

With this Werk, we close the dropdown now instead.
