# Stop AWS/RDS check from crashing if metrics are not available

key        | value
---------- | ---
date       | 2026-01-09T07:42:39+00:00
version    | 2.4.0p19
class      | fix
edition    | cre
component  | checks
level      | 1
compatible | yes

If you have previously discovered AWS/RDS services and the monitored AWS
instance is unavailable or has been disabled in the meantime the check
would crash, expecting the data.

With this Werk, the check will go stale instead.
