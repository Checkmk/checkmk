# Setup pages loaded slowly on large configurations

Previously, on installations with a very large configuration, every Setup
page took noticeably longer to load - in extreme cases around 10 seconds per
navigation step.

The product usage analytics popup is evaluated on every Setup page. To decide
whether to show it, it determined whether you had already made a decision about
sharing product usage analytics. Doing so triggered a full parse of the base
configuration. On large setups, parsing the entire configuration on each
navigation step dominated the page load time.

The popup now reads only the small setting it actually needs for this decision,
so Setup pages load as fast as before regardless of the configuration size.
