# Setup pages loaded slowly on large configurations

If you have a large configuration, you may have noticed that every page in
Setup took a long time to open - in extreme cases around 10 seconds for each
step while navigating.

This affected all Setup pages. Each one ran an unnecessary, expensive check
while preparing the product usage analytics notification, and the larger your
configuration was, the more time this check added to every page load.

This has been fixed. Setup pages now open quickly again, no matter how large
your configuration is. The change is purely about performance and does not
affect your configuration or the product usage analytics settings.
