# OTel: Fix Receiver/prometheus collector configurations in distributed setup

---------- | ---

version    | 2.5.0b1

In a distributed setup, a bug previously caused the OpenTelemetry Collector configuration to be overwritten across different sites.
With this change, the OpenTelemetry Collector configs for GRPC/HTTP receivers and Prometheus scrapers are now only modified in the specified sites.
