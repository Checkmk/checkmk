# Fix showing group alias instead of name/ID on the central site

key        | value
---------- | ---
date       | 2026-01-28T11:44:16+00:00
version    | 2.5.0b1
class      | fix
edition    | cme
component  | multisite
level      | 1

If a customer specific host was assigned to a host group for which the customer
field was set to another customer (incl. `Provider`) then the host group name
(resp. ID) was shown instead of the alias. This has been fixed for central sites.
