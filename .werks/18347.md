# Clustered services: Fix for disabling services

key        | value
---------- | ---
date       | 2025-12-01T14:29:36+00:00
version    | 2.4.0p18
class      | fix
edition    | cre
component  | checks
level      | 1
compatible | yes

On the service discovery page of a cluster host users are offered a "Move to disabled services" button, which had no effect.

This change allows users again to disable clustered services from the service discovery page.

No manual action is required from users to benefit from this fix.
