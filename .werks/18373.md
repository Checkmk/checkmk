# Also allow connections from Ultimate to Pro/Community editions

key        | value
---------- | ---
date       | 2026-03-04T13:18:24+00:00
version    | 2.5.0b1
class      | feature
edition    | ultimate
component  | multisite
level      | 1
compatible | yes

Central sites running an Ultimate edition can now also connect to remote sites with a Pro/Community edition.
The only restriction is that the remote site must be connected without configuration replication.
