# Fixed bug where site specific global settings were no longer editable on the central site

When the last remote site was removed from a distributed monitoring setup, the site-specific global settings button disappeared for the central site as well. This made any previously configured site-specific settings on the central site inaccessible and uneditable.
The button does now only vanish if a central site has no site specific settings set.
