The windows and linux netstat command return slightly different names for the
same meaning. [Werk 19201](https://checkmk.com/werk/19201) already added some
new translation entries to make sure all windows names can be translated to the
linux names, which we use in our code.

This Werk adds two new translations for the German entries `FIN_WARTEN_1` and
`FIN_WARTEN_2` and one windows-linux translations for `LISTEN`.
