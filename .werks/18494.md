# Grafana endpoint accidentally triggered configuration update

An internal Grafana REST-API endpoint triggered an update of the internal
configuration counter. If a Grafana dashboard was open and at the same time
someone wanted to activate changes the following error could happen:

```
cmk.ccc.exceptions.MKGeneralException:
The configuration was changed during activation.
Terminating this activation to ensure configuration integrity.
Please try again.
```
