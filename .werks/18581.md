# Dashboard: config file format changed

The internal storage format of dashboards has changed. Dashboard widgets are now stored with stable, unique IDs instead of being kept in an ordered list. Additionally, view widgets now store their view configuration separately from the widget configuration itself.

Existing dashboards are migrated during the update. MKPs are migrated on load, but not edited on disk. Dashboard MKPs created with this or newer versions of Checkmk are not compatible with older versions.
