# Allow single-row static text widgets in responsive dashboard grid

key        | value
---------- | ---
date       | 2026-03-26T12:57:43.953827+00:00
version    | 2.5.0b3
class      | feature
edition    | community
component  | multisite
level      | 1
compatible | yes

Static text widgets in the responsive dashboard grid layout can now be as small as a single row, making them useful as compact labels or separators.

To support this, the grid row height was increased and the default minimum widget height was reduced. Existing dashboards using the responsive grid layout may look slightly different after this change.
