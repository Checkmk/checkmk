# Warn when editing MKP without version change

key        | value
---------- | ---
date       | 2026-04-01T13:36:51+00:00
version    | 2.5.0b3
class      | feature
edition    | pro
component  | wato
level      | 1
compatible | yes

The extension package editor now displays a confirmation dialog when users save an MKP without increasing its version number.
This dialog requires users to explicitly confirm their intention to save the changes without a version bump, helping to prevent accidental updates.

Previously, such changes were applied silently, which could lead to unexpected behavior during package distribution or updates.
With this new confirmation step, users gain clearer control over modifications, reducing the risk of unintended consequences when editing and distributing extension packages.
