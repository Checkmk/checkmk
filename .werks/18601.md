# Remove deprecated 'kernel' check plugin

edition    | community

The deprecated `kernel` check plugin, which produced one service per kernel
counter (e.g. _Kernel Context Switches_, _Kernel Major Page Faults_,
_Kernel Process Creations_), has been removed.

Werk #8969 announced this deprecation in 2.0.0i1 and introduced the
consolidated _Kernel Performance_ service as a replacement, which remains
available and covers the same data.
