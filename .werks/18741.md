# OTel: Rename Application Monitoring to Telemetry

To make your integration setup more intuitive, we’ve reorganized our monitoring settings to better support OpenTelemetry workflows:

- "Application monitoring" is now "Telemetry": You’ll find this updated name in the Setup menu and in Global Settings.

- The OTel Collector receiver and Prometheus Scraper configurations have been moved from the "Hosts" section in Setup to the new "Telemetry" section.
