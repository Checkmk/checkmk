# Voltage levels (UPS / PDU / other devices): allow alerting on over-voltage

The *Voltage levels (UPS / PDU / Other Devices)* ruleset previously only
exposed lower threshold fields (warning / critical if below). It was therefore
not possible to alert on abnormally high voltage readings reported by UPS,
PDU and similar devices.

The ruleset now also exposes optional upper threshold fields (warning /
critical at or above), matching the way the *Frequency* parameter behaves in
related rulesets. The new fields are optional and default to unset, so
existing rules and check results are unchanged unless an administrator
explicitly configures upper thresholds.

The following check plugins evaluate the new upper thresholds:

* `ups_in_voltage`
* `ups_out_voltage`
* `ups_socomec_in_voltage`
* `ups_socomec_out_voltage`
* `ipr400_in_voltage`
* `emc_isilon_power`
