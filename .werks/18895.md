# Fix temperature thresholds of Aruba 6100+ switches

The explicit default temperature thresholds for the Aruba 6100+ series have been raised across specific sensor limits to better reflect the actual hardware tolerances, see the man page for an overview.
This adjustment reduces false positives on components designed to run at higher temperatures.
Users who wish to retain the previous, lower thresholds must now configure them manually via the corresponding temperature monitoring rules.
