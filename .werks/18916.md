# Improvement to our 503 site not start page

The Checkmk site Apache error page has been redesigned to use a dark theme that matches the Checkmk web interface. This improved error page makes it easier for site operators to recognize when the site is down and provides clear instructions to start it, reducing confusion during downtime.

The new error page includes the following features: a dark theme consistent with the Checkmk interface, the Checkmk logo and favicon, clear instructions on how to start the site using `omd start`, automatic refresh every 60 seconds, and a responsive layout that adapts to all screen sizes.

This update is purely cosmetic and does not affect any functionality. The page is fully self-contained with inline CSS and SVG, as no site assets are available when the site Apache is down.

To apply this redesign on existing sites, run the following command:
```
omd update-apache-config SITE
```
