# Move Customer Identifier string to SAN Extension

In Checkmk Ultimate MT (MSP), we have moved the Customer Identifier string from the Common Name (CN) of the Customer Broker Certificate Authority to the Subject Alternative Name (SAN) extension. This change resolves the previous character limit on the X.509 Common Name, which formerly prevented customers with long identifiers from generating broker certificates.

Every new broker certificate created within Checkmk will automatically include this property.
