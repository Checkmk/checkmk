# Fix Customer Broker Certificate creation

---------- | ---

version    | 2.4.0p23
class      | fix
edition    | cme
component  | core
level      | 1
compatible | yes

In Checkmk MSP, the Customer Broker Certificate creation process has been updated to include the customer's name abbreviation along with a corresponding hash in the Common Name.
This change resolves the previous character limit on the X.509 Common Name, which formerly prevented customers with long identifiers from generating broker certificates.

Every new broker certificate created within Checkmk will automatically include this property.
