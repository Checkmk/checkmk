# infoblox_service: Fix support for NIOS 9.X

key        | value
---------- | ---
date       | 2025-11-27T11:07:22+00:00
version    | 2.4.0p17
class      | fix
edition    | cre
component  | checks
level      | 1

With infoblox NIOS v9 the `IB-PLATFORMONE-MIB::IBServiceName` have
changed. Specifically the LCD service with ID 13 has been removed.
We use these name as service items. Please run a re-discovery on the
affected hosts.
