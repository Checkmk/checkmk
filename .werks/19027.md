# netscaler_cpu: Fix Average configuration of CPU utilization of Devices with Modules ruleset

key        | value
---------- | ---

version    | 3.0.0b1
class      | fix
edition    | community

The `netscaler_cpu` check plugin now respects the `Average` configuration provided by the `CPU utilization of Devices with Modules` ruleset.

Previously, it reported inaccurate CPU metrics and triggered incorrect alerts. No changes to your existing rules or configurations are required to benefit from this fix.
