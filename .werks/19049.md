# Fixed Livestatus crash when the rrdcached is not running

key        | value
---------- | ---

version    | 2.4.0p21

Checkmk 2.4.0 introduced a regression where retrieving metric data via
Livestatus results in monitoring core crash when the rrdcached is not
running. This has been fixed, we now just log an error.
