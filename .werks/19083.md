# Prevent crash when showing notes URL for hosts or services

key        | value
---------- | ---

Using the “Notes (URL) for Services” or “Notes (URL) for Hosts” columns in
views could previously lead to a crash with an error such as "KeyError:
'host_address'".

This issue has been fixed, and the notes URL columns can now be used safely
without causing a crash.
