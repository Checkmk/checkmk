# Agent Bakery signing key expiry notification now uses BCC for recipients

key        | value
---------- | ---
date       | 2026-02-26T13:44:47.858076+00:00
version    | 3.0.0b1
class      | fix
edition    | pro
component  | agents
level      | 1
compatible | yes

When Checkmk sends notification emails about expiring Agent Bakery signing
keys, all recipient addresses were previously listed in the `To` header.
In multi-tenant environments this caused all recipients, including
customers from different organizations, to see each other's email
addresses.

Recipients are now placed in the `Bcc` header instead, so that individual
addresses remain hidden from all other recipients.
