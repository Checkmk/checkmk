# Test notifications: Respect host tags

key        | value
---------- | ---

class      | fix
edition    | community

Notification rules with host tag filters produced incorrect results during
testing because the host tags were not computed correctly. As a result, users
might have experienced wrong test results.

This issue has now been resolved by ensuring that host tags are computed
correctly during notification testing.
