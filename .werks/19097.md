# Fix slow "Total Sent Notifications" link due to missing time filter

Previously, clicking the "Total sent notifications" link on the notification
configuration page resulted in unexpectedly long load times — even for a small
number of notifications. The link was missing the time range unit parameter, so
the 7-day time filter was silently ignored and the entire notification log
history was loaded instead.
