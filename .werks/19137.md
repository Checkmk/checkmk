# Datadog: Don't crash on Datadog server error

Occasionally it could happen that the Datadog API server responds with a error instead of returning data correctly.
In this case the special agent would crash and create a crash report.

This Werk checks for server errors and handles the server error by returning the error code and reason while correctly shutting down the special agent.
