# ibm_svc_host: critical state no longer overridden by warning threshold

key        | value
---------- | ---
date       | 2026-05-13T08:53:45+00:00
version    | 2.6.0b1
class      | fix
edition    | cre
component  | checks
level      | 1

When both warn and crit thresholds were set and a value met the crit threshold,
the plugin would overwrite the CRIT state with a WARN state.
Changed so CRIT takes priority.
