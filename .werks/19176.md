# Empty Rows Removed From Alert Statistics View

key        | value
---------- | ---
date       | 2026-01-26T09:08:51+00:00
version    | 2.4.0p21
class      | fix
edition    | cre
component  | multisite
level      | 1
compatible | yes

On fresh sites with no added hosts, or with sites which have logs but no hosts yet, the Alert statistics view, and derived reports, would have an extra empty row with no data.

This fix makes sure only valid data appears.

No manual intervention is required.
