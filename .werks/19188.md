# Reduce locking of configuration

key        | value
---------- | ---
date       | 2026-02-24T12:26:26+00:00
version    | 2.4.0p23
class      | feature
edition    | cee
component  | dcd
level      | 2
compatible | yes

In some configurations with extensive use of multiple connectors, the configuration could become locked, causing the GUI to freeze and become unresponsive.

Since this release, the REST API for DCD has been changed to prevent configuration locking, thereby resolving this issue and ensuring the GUI remains responsive.

No user actions are required.
