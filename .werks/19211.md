# Deploy signed PowerShell plugins with Windows agent

The Windows agent MSI package did not include signed variants of the
PowerShell-based agent plugins. Users who enforced PowerShell execution policies
had to sign the plugins manually or lower the execution policy site-wide.

Starting with this release, the MSI package ships pre-signed PowerShell plugins.
The bakery prefers the signed variant over the unsigned one when both are
present, so existing setups benefit automatically after an agent update.
