# mk-sql: Command line switch to list active local SQL Server instances on Windows

There was previously no way to quickly check which SQL Server instances on a
Windows host were currently running without connecting to each one individually.

Starting with this release, mk-sql provides the `--active-instances` command
line switch to list all active local SQL Server instances on a Windows host:

`mk-sql.exe --active-instances`
