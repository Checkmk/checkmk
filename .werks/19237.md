# Render custom graphs with scalars and/or constants only

key        | value
---------- | ---
date       | 2026-02-06T10:30:29+00:00
version    | 2.4.0p22
class      | fix
edition    | cee
component  | multisite
level      | 1
compatible | yes

Custom graphs containing only scalars and/or constants did not render.
As a result, the Checkmk UI always displayed an empty canvas, preventing users who created such graphs from viewing their data.
After this fix, these graphs now display correctly without requiring any user action.
