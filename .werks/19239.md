This werk only affects users of Synthetic Monitoring.

The RCC binaries (Linux/Windows) shipped with the Synthetic Monitoring feature are now built with Go 1.26 to address [CVE-2025-68121](https://www.cve.org/CVERecord?id=CVE-2025-68121). Checkmk was NOT affected by this vulnerability.

Users must update the agent on affected hosts to receive the fix.
