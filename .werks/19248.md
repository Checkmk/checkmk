In addition to the `x64` agent packages, we ship them now for `aarch64` as well.
Please find them pre built within your site's directory:
```
share/check_mk/agents/check-mk-agent-2.5.0-2026.04.22-1.aarch64.rpm
share/check_mk/agents/check-mk-agent-2.5.0-2026.04.22-1.noarch.rpm
share/check_mk/agents/check-mk-agent_2.5.0-2026.04.22-1_all.deb
share/check_mk/agents/check-mk-agent_2.5.0-2026.04.22-1_arm64.deb
```

This is an enhancement to [Werk #19275](https://checkmk.com/werk/19275) which announced the availablity of the Linux agent controller for `aarch64`.
