# Deprecation of Azure VM instances and Azure storage accounts dashboards

This change is due to the planned deprecation of the legacy Azure special agent in a future Checkmk release.

The 'Azure VM instances' and 'Azure storage accounts' dashboards have been marked as deprecated and hidden them from the main navigation.

We have replaced these with new dashboards powered by the Azure special agent v2. To begin using them, please set up the new special agent.

You can still access the old dashboards via the dropdown menu at the top of any active dashboard.

For more information please see the werk #19467
