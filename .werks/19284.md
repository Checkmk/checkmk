# NetApp ONTAP: Monitor S3 bucket usage

key        | value
---------- | ---
date       | 2026-04-24T10:16:11.586749+00:00
version    | 2.4.0p33
class      | feature
edition    | cre
component  | checks
level      | 1
compatible | yes

NetApp ONTAP S3 bucket usage can now be monitored. Enable the *S3 Buckets* fetch in the
*NetApp via Ontap REST API* rule and run a service discovery on the affected hosts.

Buckets are reported as *Qtree* services (named `Qtree <flexgroup>/<bucket>`), so existing
history, rules and the *NetApp Qtree discovery* ruleset apply to them as well.
