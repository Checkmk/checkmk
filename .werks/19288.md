# Kubernetes: Add swap usage monitoring for containers

key        | value
---------- | ---
date       | 2026-05-27T15:22:08.838802+00:00
version    | 3.0.0b1
class      | feature
edition    | community
component  | checks
level      | 1
compatible | yes

Upstream Kubernetes supports swap as a stable feature as of version 1.34.

Swap usage is now shown as part of the _Memory resources_ service for
Kubernetes hosts.

At the pod host level, swap usage across containers is summed and
displayed. For workload, namespace, and cluster level hosts, swap usage
is rolled up and summed, similarly to memory usage.

Configurable thresholds are available under _Kubernetes memory resource
utilization_.
