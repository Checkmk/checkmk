# Kubernetes: Respect namespace filtering for CronJobs

key        | value
---------- | ---
date       | 2026-06-05T13:58:41.815412+00:00
version    | 2.5.0p7
class      | fix
edition    | community
component  | checks
level      | 1
compatible | no

Fixed an issue that prevented Kubernetes CronJobs in non-monitored namespaces
from being filtered out when namespace filtering was enabled, causing them to
incorrectly appear in monitoring.

**Impact:** Any rules set up to monitor CronJobs in non-monitored namespaces will
no longer generate alerts until the namespace is added to monitoring.

**Recommended action:** After updating, please review the list of namespaces
monitored by the special agent to ensure all desired namespaces are included.
