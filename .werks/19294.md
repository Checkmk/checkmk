# Check Point power supplies: make service state configurable

Power Supply services for Check Point power supplies now allow users to customize service states
through the new Check Point powersupply rule.
This rule exposes two device statuses: Up (default OK) and Present (default CRIT);
any other status reported by the device defaults to CRIT.

Previously, these services reported OK only if the device status was Up, and CRIT for all other statuses.
With this change, users can tailor which device statuses trigger OK or CRIT service states.

The default behavior remains unchanged, so existing setups will continue to function without any required action.
