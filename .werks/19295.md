# Azure v1: update Azure API versions for backup and resource health endpoints

The Azure special agent has been updated to use newer API versions for two
endpoints:

- **Backup protected items**: `2022-05-01` → `2025-02-01`
  (`Microsoft.RecoveryServices/vaults/.../backupProtectedItems`)
- **Resource health**: `2022-05-01` → `2025-05-01`
  (`Microsoft.ResourceHealth/availabilityStatuses`)

This ensures compatibility with current Azure API requirements and avoids
potential failures caused by deprecated API versions.
