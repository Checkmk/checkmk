# Restrict RabbitMQ inter-site broker to TLS 1.3 only

---------- | ---

version    | 2.4.0p26
class      | fix
edition    | cre
component  | multisite
level      | 1
compatible | yes

The internal RabbitMQ message broker, used for inter-site communication,
previously accepted both TLS 1.2 and TLS 1.3 connections. It now accepts
TLS 1.3 only.
With this werk we also restrict connections to the RabbitMQ management plugin
to TLS 1.3.

TLS 1.2 permits cipher suites that are considered weak by modern security
standards. TLS 1.3 removes all such cipher suites by design.
