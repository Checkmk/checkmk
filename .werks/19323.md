# Custom graph: The timeframe adjustment is only retained during the editing session or when switching from the related graph view.

key        | value
---------- | ---
date       | 2026-03-24T11:13:49.307789+00:00
version    | 2.5.0b2
class      | fix
edition    | pro
component  | multisite
level      | 1
compatible | yes

In the custom graph editor page, you can adjust the graph's time axis by
dragging it with the mouse. Previously, this dragging remained even after
exiting edit mode. However, if you later return to add new metrics, they might
not be visible because they fall outside the timeframe.

Now, moving the time axis is only persisted in edit mode or when coming from
the custom graph view page.
