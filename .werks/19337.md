# hitachi_hnas_cpu: Fix Average configuration of CPU utilization of Devices with Modules ruleset

The `hitachi_hnas_cpu` check plugin now respects the `Average` configuration provided by the `CPU utilization of Devices with Modules` ruleset.

Previously, it reported inaccurate CPU metrics and triggered incorrect alerts. No changes to your existing rules or configurations are required to benefit from this fix.
