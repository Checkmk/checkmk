# Fix dashboard crashes after update due to incompatible graph size type

key        | value
---------- | ---
date       | 2026-02-19T14:04:52+00:00
version    | 2.5.0b1
class      | fix
edition    | community
component  | multisite
level      | 2
compatible | yes

Rendering of graphs that had been resized could lead to dashboard crashes showing a stacktrace due to incompatibilities in the storage format for the graph sizes.

This has been fixed and dashboards with graphs will function normally after updates without requiring any manual reconfiguration. Users do not need to take any action, as the fix automatically maintains seamless operation of all existing graph configurations.
