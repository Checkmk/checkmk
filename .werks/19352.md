# Custom graph editor: Allow empty symbol for custom units

key        | value
---------- | ---
date       | 2026-05-22T13:22:28.547898+00:00
version    | 2.5.0p4
class      | fix
edition    | community
component  | multisite
level      | 1
compatible | yes

Werk #19347 introduced a validation that rejected an empty unit notation
symbol for non-time custom-unit notations in the Custom graph editor.
The symbol of a custom unit is optional, however (for example for
dimensionless metrics), so that validation was incorrect.

The check has been removed; empty symbols are again accepted for non-time
custom-unit notations. The other validations introduced by werk #19347
(precision digits, constant graph line value, transformation percentile)
remain in place.
