# Forecast graphs: hide non-functional "Add to ..." option

key        | value
---------- | ---
date       | 2026-06-10T00:00:00+00:00
version    | 3.0.0b1
class      | fix
edition    | pro
component  | metrics
level      | 1
compatible | yes

Previously, forecast graph pages showed an "Add to ..." popup icon offering
to add the graph to a dashboard or to a custom graph. Forecast graphs
cannot actually be added to either, and attempting to do so failed.

The icon is no longer rendered on forecast graph pages.
