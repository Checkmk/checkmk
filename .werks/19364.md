# Fix wrong inline help shown on host tag dropdowns

Previously, the inline help shown next to host tag dropdowns on the host
configuration page could display the wrong text — often a help text
configured on a different, unrelated tag group. In the worst case, the help
text from a single custom tag group appeared on every host tag dropdown,
including built-in tag groups.

The stored tag configuration was never affected.
