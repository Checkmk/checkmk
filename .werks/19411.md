# "Use new service names" global setting now uses titled sections per plugin

The global setting "Use new service names" under "Global settings > Execution of checks" has been reworked.

Previously, this setting displayed a flat list of checkboxes. The setting now displays each plugin as a titled section with a "Use the new service name" checkbox underneath. This change was necessary due to a change in the underlying data structure to allow Checkmk to tell apart plugins that were explicitly disabled from plugins that were newly added and not yet configured.

Existing configurations are migrated automatically. No manual action is required.
