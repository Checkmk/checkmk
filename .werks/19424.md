# Fix notifications search query disappearing

The search/filter input on the notification rules list page was reset to
empty after performing any of the following actions:

- Deleting a rule
- Reordering (dragging) a rule
- Opening the edit or clone page for a rule
- Saving an edited rule (returning to the list)
- Pressing the "Notifications" breadcrumb link from the edit page
- Pressing "Cancel" on the quick-setup edit page

The filter is now preserved consistently across all these actions, for
global notification rules, user-specific notification rules, and personal
notification rules.
