When editing a notification rule, a "Save" button is now shown. 
It saves the notification and returns to the notifications page.
The buttons to  "Apply & go to notifications" and "Apply & go to overview" 
are still available.

The new button is not present when creating a new notification so that the
user is guided to test or create another rule and test right after.
