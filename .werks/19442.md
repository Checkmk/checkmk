# cisco_wlc and wlc_clients: Support Cisco 9800H1

This Werk adds support for the Cisco 9800H1 wireless controller to both the
`cisco_wlc` check plugin as well as the `wlc_clients` check plugin.

The new services will be available upon performing service discovery on the
host.
