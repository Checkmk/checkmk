# Kubernetes 1.34 is now officially supported

key        | value

As of this release of Checkmk, Kubernetes 1.34 is now officially supported.

The following versions of Kubernetes are supported:

```
Checkmk 2.3: 1.24, 1.25, 1.26, 1.27, 1.28, 1.29, 1.30, 1.31 (no change)
Checkmk 2.4: 1.26, 1.27, 1.28, 1.29, 1.30, 1.31, 1.32, 1.33, 1.34
Checkmk 2.5: 1.31, 1.32, 1.33, 1.34
```

As official support for Kubernetes 1.30 has been dropped from Checkmk 2.5,
we mark this Werk as incompatible.

For more details on the supported versions policy please refer to
[Werk #19449](https://checkmk.com/werk/19449).
