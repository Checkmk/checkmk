# Azure overview dashboard: fix empty Tenants "Azure agent info" column

key        | value
---------- | ---
date       | 2026-04-01T13:26:15.978685+00:00
version    | 2.5.0b3
class      | fix

Previously, the _Azure agent info_ column in the Tenants view of the Azure
overview dashboard was displayed empty. The cause was a service name
mismatch: the Azure v2 special agent uses _Azure agent info_ (sentence case),
but the dashboard's join configuration still referenced the old name
_Azure Agent Info_ (title case). This has been fixed.
