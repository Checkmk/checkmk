# Activate changes slideout: modify ui when there are no pending changes

The Activate Changes slideout has been updated to improve clarity when there are no pending changes to activate.

In this case, the sites section now displays the message: "No pending changes on your site(s)".

Additionally, the changes section is disabled and shows the message: "Changes on selected sites (0 of 0)". This explicitly informs users that no action is required, preventing any confusion about pending changes.
