# Activate changes slideout: show success together with warning/error messages

Slideout activation now shows a success message for sites that were activated successfully, including the number of such sites. This means you can quickly see which sites activated without issues, even if some encountered warnings or errors. Please bear in mind that sites that were activated with warnings will be counted as successfully activated sites, because they are indeed activated and working, albeit with some issues that should be addressed.

Additionally, any warning or error messages will disappear after reopening the slideout because the status bar already reflects the current warning/error status. This prevents confusion from outdated messages.
