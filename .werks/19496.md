# Fix wording in agent slideout registration instructions

Previously, the agent connection test slide-out panel told you to run the
registration command "after you have downloaded the agent". Since the agent
must first be installed before registration is possible, the wording has been
corrected to "after you have installed the agent" for Windows, Linux, and AIX
hosts. The Solaris instruction has also been updated to clarify it triggers
registration rather than installation.
