# Fix dialog background color in light theme

key        | value
---------- | ---
date       | 2026-03-20T13:19:26.739237+00:00
version    | 2.5.0b2
class      | fix
edition    | community
component  | wato
level      | 1
compatible | yes

Dialog panels in the quick activation of changes used the same background color as their parent component in the light theme, making their borders invisible. This could cause difficulty in distinguishing the panels, potentially leading to confusion or misclicks.

We changed the background color to ensure the dialog panel borders are clearly visible, improving usability. Users do not need to take any action to benefit from this change.
