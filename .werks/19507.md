# Simplify page menu search reset process

Clearing up the search query in the "Notifications" page was 
not possible unless the user clicked on a grayed-out X.

This change improves the user experience by allowing
the user to clear the search when searching for an empty string.

Additionally, the button to reset the search is no longer grayed-out.
