# Tabs and toggle buttons overflow at narrow browser widths

key        | value
---------- | ---
date       | 2026-03-23T10:58:06+00:00
version    | 2.5.0b2
class      | fix
edition    | community
component  | multisite
level      | 1
compatible | yes

The tab navigation (e.g. Windows/Linux/Solaris/AIX in the agent installation
slideout) and toggle button groups (e.g. RPM/DEB/TGZ) did not adapt to narrow
browser widths, causing horizontal overflow instead of wrapping to the next
line.

Both components now use flex-wrap to gracefully handle narrow viewports.
