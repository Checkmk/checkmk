# Fix empty host/service group columns in CSV and JSON export

Previously, when exporting a view containing the "Host groups the host is
member of" or "Service groups the service is member of" columns to CSV or JSON,
the exported values were empty or contained only commas instead of the actual
group names. This was caused by a refactoring of the painter link generation
which introduced a display options check. During CSV/JSON export, display
options disable link rendering, causing the link helper to return empty HTML.
Since these painters relied entirely on link rendering for their output and had
no dedicated export methods, the exported data was empty.

The painters now provide explicit export methods that return group names
directly from the row data, bypassing HTML link generation.
