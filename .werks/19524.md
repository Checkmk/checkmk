# Update time-specific parameters button rendering

key        | value
---------- | ---
date       | 2026-03-20T13:17:58.651344+00:00
version    | 2.5.0b2
class      | feature
edition    | community

The rendering order of the toggle button for time-specific parameters has been updated. The
"Enable time-specific parameters" and "Disable time-specific parameters" buttons now appear
**above** the associated form fields rather than below them.

This brings the button placement in line with the broader form rendering improvements introduced
in Checkmk 2.5 ([Werk #19167](https://checkmk.com/werk/19167)) and 2.6, where actions that control a section of a form are displayed before the
content they affect. The change applies to all rulesets that use time-specific parameters.
