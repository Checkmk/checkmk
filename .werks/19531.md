# Fix access to Email check related ruleset without OAuth2 connections permission

---------- | ---

class      | fix
edition    | community

With the introduction of OAuth2 connections users without the
permission "Manage OAuth2 connections" were no longer able to access
the email check related rulesets.

With this change this is again possible.
