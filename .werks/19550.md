# Fix Gerrit special agent cache implementation

key        | value
---------- | ---
date       | 2026-03-25T12:22:14.770909+00:00
version    | 2.5.0b3
class      | fix
edition    | community
component  | checks
level      | 1
compatible | yes

The cache implementation used in the Gerrit special agent plugin would lead to
the cache being invalidated after each run. This fix now allows the cache
implementation to run as it should. There is nothing for the user to do outside
of updating to the latest version of Checkmk since the cached data will be
overwritten.
