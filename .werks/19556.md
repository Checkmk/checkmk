# Fix Redfish agent error on BMCs that omit Systems from service root

---------- | ---
date       | 2026-03-31T14:55:48.525078+00:00
version    | 2.6.0b1
class      | fix
edition    | community
component  | checks
level      | 1
compatible | yes

Previously, the Redfish special agent reported "probably not a Redfish computer
system - missing Systems or Chassis information" on certain BMCs (e.g. LITEON)
that don't advertise the `Systems` or `Chassis` endpoints in their `/redfish/v1`
service root response — even though those endpoints exist and respond normally.

The agent now probes the standard paths `/redfish/v1/Systems` and `/redfish/v1/Chassis`
directly before giving up.
