# Inventory widget: offer all available attributes

key        | value
---------- | ---
date       | 2026-04-01T09:24:30.794095+00:00
version    | 2.6.0b1
class      | feature
edition    | pro
component  | multisite
level      | 1
compatible | yes

You can now select any registered HW/SW inventory attribute when configuring the
inventory dashboard widget. Previously, only a hardcoded list of about 20 attributes
was available.

The list of available attributes is derived dynamically from inventory display hint
plugins, so it automatically includes new attributes whenever inventory UI plugins are
added — no manual updates needed. Edition-specific attributes (e.g. Azure or Kubernetes
metadata) are included when the corresponding plugins are present.
