# Fix widget titles not shown for non-admin dashboard viewers

key        | value
---------- | ---
date       | 2026-04-02T07:01:35.415547+00:00
version    | 2.6.0b1
class      | fix
edition    | community
component  | rest-api
level      | 1
compatible | yes

Previously, users without the *Edit dashboards* permission (e.g. guest role users)
could not see widget titles on dashboards. The titles appeared blank while the widget
content rendered correctly.

The REST API endpoint for computing widget titles incorrectly required the
`general.edit_dashboards` permission, which is an edit-level permission not
granted to dashboard viewers.
