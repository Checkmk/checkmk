# Disambiguate duplicate titles in inventory widget dropdown

key        | value
---------- | ---
date       | 2026-04-07T14:41:01.200406+00:00
version    | 2.6.0b1
class      | fix
edition    | pro
component  | multisite
level      | 1
compatible | yes

When configuring the HW/SW inventory dashboard widget, some attributes from
different inventory paths shared the same display title (e.g. three entries
all showing "Metadata ➤ Object" for Azure, Kubernetes, and Proxmox VE).

The dropdown now detects duplicate titles and prepends the parent category
to distinguish them (e.g. "Azure ➤ Metadata ➤ Object").
