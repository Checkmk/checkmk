# Fix service status display for single metric dashboard widget

key        | value
---------- | ---
date       | 2026-04-09T06:57:53.176326+00:00
version    | 2.5.0b5
class      | fix
edition    | pro
component  | multisite
level      | 1
compatible | yes

Previously, the "Service status" setting in the single metric dashboard widget had no effect.
While the setting was visible in the widget wizard, it was not wired up to the widget content,
so the service status was never displayed regardless of the configuration.

This has been fixed. The single metric widget now offers the same service status options as
the gauge widget: a colored status label (for all or only non-OK states) and an optional
colored metric background.
