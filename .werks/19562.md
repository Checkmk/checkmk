Added several check plugins for checking the sensors of a Room Alert 3S Environment Monitor via SNMP.

The following can be checked:

* The internal temperature reading
* The temperature from the digital sensor
* Any one of the following readings depending on the hardware configuration:
    * The humidity from the digital sensor, if available
    * The voltage from the digital sensor, if available
    * The power state from the digital sensor, if available

As this is a new plugin, no manual intervention is required for existing Checkmk installations.
