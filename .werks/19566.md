# Event Console Will Continue to Translate OIDs Even with Missing MIBs

key        | value
---------- | ---
date       | 2026-04-14T05:41:20.373668+00:00
version    | 2.4.0p27
class      | fix
edition    | cre
component  | ec
level      | 1
compatible | yes

When uploading custom MIBs for use with the Event Console, the UI may warn about missing dependencies not imported for these MIBs.

However, when the Event Console starts, it will receive traps but not translate any OIDs because not all of the MIBs could be processed.

With this change, the Event Console will make use of any MIBs which are usuable, thus still being able to translate known OIDs.

No manual intervention is required.
