# Remove non-functional 10-second interval from Event Console rules

key        | value
---------- | ---
date       | 2026-04-15T07:54:02.198296+00:00
version    | 2.6.0b1
class      | fix
edition    | community
component  | ec
level      | 1
compatible | yes

Previously, the _Expect regular messages_ setting in _Event Console_ rules offered
10 seconds as the lowest interval. However, this interval has not functioned
correctly in any recent Checkmk release, so it has been removed.

The new minimum interval is 1 minute.

Any existing rules that use a 10-second interval will be changed to 1 minute instead during update.
