# Fix activate changes slideout result not updating

key        | value
---------- | ---
date       | 2026-04-01T15:05:32.269022+00:00
version    | 2.6.0b1
class      | fix
edition    | community
component  | multisite
level      | 1

Whenever a change was activated in the quick activation
slideout, the slideout status bar would show outdated
information from the previous activation, until the
slideout is closed and reopened.

This is now fixed, so that the resulting quick activation
slideout state reflects the latest activation result.
