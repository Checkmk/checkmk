# Quick activation content hidden in small screens

edition    | community

When the screen height is small, or the zoom level is high enough,
it is possible that the list of changes in the "Quick activation
of changes" popout is hidden and can not be scrolled into.

This is now fixed by adding a higher level scrollbar for the popout
content, that only appears when the popout content doesn't fit the screen.

The scrollbar for the list of changes is still present and its
functionality should not be affected.
