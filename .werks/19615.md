# Downtimes: include the author's username in the comment

key        | value
---------- | ---
date       | 2026-03-31T05:26:34.112713+00:00
version    | 2.5.0b3
class      | feature
edition    | community
component  | multisite
level      | 1
compatible | yes

When scheduling a downtime, the comment now automatically includes the
username of the user who created the downtime in the format
`Author (<username>): <comment>`. This makes it easy to identify who
scheduled a downtime directly from the comment field without needing
additional tools or audit logs.
