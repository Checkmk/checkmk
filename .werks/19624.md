# Fix Customer filter being ignored in subreport elements

key        | value
---------- | ---
date       | 2026-05-25T10:27:50.396246+00:00
version    | 2.4.0p31
class      | fix
edition    | cme
component  | reporting
level      | 1
compatible | yes

Previously, when a Customer filter was configured on a report containing
_Host report for multiple hosts_ or _Service report for multiple services_
elements, the filter had no effect. The subreport elements would enumerate
hosts and services from all customers, ignoring the configured Customer filter.

This has been fixed. Subreport elements now correctly scope their host and
service queries to the selected customer.

As a workaround, a Site filter selecting the customer's site(s) could be used
to achieve the same result.
