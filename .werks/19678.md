During the Beta release we've received some feedback from users regarding the
usefulness of the ordering. A collection of fixes and improvements were made to
improve the ordering of the results.

The main improvements are as follows:

- Promoting important topics to the top and demoting topics like "Enforced
  services", "Global settings", and "Deprecated rule sets" to the bottom of the
  results window.
- Grouping related topics closer together.
- Fixing the sorting of topics for non-English environments.
- Pushing query matches to URL below matches on the title.

As this is only a change to the underlying sorting algorithm, there is no user
intervention required to get these benefits. If you have any suggestions to
improve the current state of the search, feel free to reach out to the forum
with suggestions.
