# checkpoint_powersupply: Recognize "OK" as valid power supply status

Previously, the _Power Supply_ check for Check Point firewalls only recognized "Up" as a healthy
status. Some devices report "OK" instead, which was treated as `CRIT` because it wasn't in
the default status mapping.

"OK" is now recognized and maps to `OK` by default. You can also configure the monitoring state
for this status via the rule _Check Point powersupply_.
