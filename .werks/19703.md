# apache-omd: Ignore zombie processes on shutdown

key        | value
---------- | ---
date       | 2026-04-24T08:49:54.611341+00:00
version    | 2.6.0b1
class      | fix
edition    | community
component  | omd
level      | 1
compatible | yes

`omd stop apache` now ignores zombie processes. In particular, it no longer reports that Apache is running, if it encounters a zombie process.
This is consistent with how `omd` handles site processes.
