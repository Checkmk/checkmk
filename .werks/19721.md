# REST API: removal of previously deprecated endpoints and fields

With this Werk, previously deprecated REST API endpoints and fields are removed.

The following endpoints are affected, see their respective Werks for details:

* [Werk 17003](https://checkmk.com/werk/17003): `/domain-types/host/collections/all`
* [Werk 17512](https://checkmk.com/werk/17512): `/objects/host/{host_name}/collections/services`
* [Werk 17512](https://checkmk.com/werk/17512): `/domain-types/service/collections/all`

Additionally, the deprecated `owned_by` field on the password endpoints has been removed.
Use `editable_by` instead, see [Werk 17274](https://checkmk.com/werk/17274) for details.

The removal of the host and service endpoints also removed the REST API export links
(_Query hosts resource_, _Query services resource_) from the monitoring views in the UI.
