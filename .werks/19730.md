# New: Monitor hosts in segregated networks with the Checkmk Relay

Checkmk can now monitor hosts in segregated or isolated networks using the new Relay System.

You deploy a lightweight relay container in the segregated network.
The relay reaches out to your central Checkmk site (outbound only — no inbound firewall changes required), collects monitoring data from local hosts, and forwards results back.
Relay-monitored hosts appear just like any other host in your Checkmk monitoring.

The relay supports SNMP and special-agent checks.
Configuration automatically syncs when you activate changes, and the relay self-updates when your Checkmk site is upgraded.

To get started, go to _Setup > Hosts > Relays_ where you'll find an installation guide.
See the [Checkmk documentation](https://docs.checkmk.com/master/en/relay.html) for more details.
