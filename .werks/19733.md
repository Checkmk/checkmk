# Fix logwatch setup rules and pattern analyzer in Cloud edition

Previously, in the Cloud edition, two logwatch-related Setup pages were not
registered:

- The rules _Log file patterns_ and _Log file grouping_ were not available
  for configuration.
- Clicking _Log file pattern analyzer_ in the Setup would show an error page.

Both issues have been fixed.
