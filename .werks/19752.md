# Hide "Install & register agent" button for auto-registered hosts

Previously, the "Install & register agent" button was shown for all hosts in
push mode, including hosts that were automatically created via push
auto-registration. For these hosts, the agent is already installed and actively
sending data, making the button misleading.

The button is now hidden for auto-registered hosts, identified by the
`cmk/agent_auto_registered` host label.
