# Notifications: translate notification method edit side panel

When the Checkmk GUI was configured to a language other than English
(for example German), opening the notification method editor from a
notification rule showed a side panel where only some texts of the page were
translated correctly.

With this werk the side panel is displayed in the language selected by
the logged-in user.
