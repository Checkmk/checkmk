# Fixed exception when saving a host on a distributed remote site

Saving a host or folder in a distributed remote site caused an error in case the site attribute was not known.
Please note that it is generally not a good idea to save a host or folder at a distributed site,
as the configuration will be overwritten the next time you click ‘Activate changes’ on the central site.
