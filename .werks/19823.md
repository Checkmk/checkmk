# Limit host and service label autocompleter to host and service labels

key        | value
---------- | ---
date       | 2026-05-21T08:36:54.807712+00:00
version    | 2.6.0b1
class      | feature
edition    | community
component  | wato
level      | 1
compatible | yes

When configuring host or service labels in Checkmk the suggested entries contain
both existing host and service labels at the same time.

With this change, we limit the suggestions to service labels when configuring service
labels and vice versa.
