# Highlight keyboard focus visually

This Werk introduces a border around the currently active input
element when navigating through forms using the keyboard via the Tab key.

Similar to Werk #19167, this change is part of a greater UI refactoring
effort and will not affect all parts of Checkmk. It is limited to rules
migrated to the `cmk.rulesets.v1` API as well as a number of pages like
the ones in "Quick Setup" or "Dynamic host management".
