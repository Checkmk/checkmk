# Kubernetes Pod resources: Allow setting alert thresholds on pending and failed pods

key        | value
---------- | ---
date       | 2026-05-20T12:45:02.244798+00:00
version    | 2.6.0b1
class      | feature
edition    | community
component  | checks
level      | 1
compatible | yes

Levels can now be set for the Kubernetes _Pod resources_ service to allow for
alerting on the number of pending and/or failed pods.

The levels can be set from the ruleset _Kubernetes pod resources_.
