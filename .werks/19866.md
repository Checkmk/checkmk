# Kubernetes: New Perf-O-Meters for several services

New Perf-O-Meters have been added for the following Kubernetes services:

- _Info_ (age)
- _Containers_ (running)
- _Pod resources_ (running)
- _Replicas_ (ready)

A new metric, `kube_info_age`, is now emitted for _Info_ services and can
be used in custom graphs and dashboards.
