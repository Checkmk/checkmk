# Agent installation slideout: clearer wording with download fallback

Werk [#19780](https://checkmk.com/werk/19780) replaced the unusable token-based
download commands in the agent installation slideout with a single warning
message when the user lacks the permissions to download from the Agent Bakery.
The message was identical across editions and offered no actionable
alternative.

The slideout now renders an edition-specific fallback in the *Linux* tab that
points the user at a working download path.
