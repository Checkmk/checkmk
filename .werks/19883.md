# Swagger UI: Fix yaml download

Downloading the OpenAPI specification in YAML format (`openapi-swagger-ui.yaml`)
from the REST API Swagger UI returned HTTP 404.

The download link for the YAML specification is accessible via
Help > REST API > Version 1 Interactive GUI. The route for this file was missing
from the versioned API registration, causing the 404 response.
