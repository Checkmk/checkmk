# Fix crash in Jenkins Memory check when metric value is not a number

Previously, the _Memory_ service on Jenkins hosts could crash when Jenkins
reported a "NaN" string value for a JVM memory pool metric. This has now been
fixed. Non-numeric memory values are now silently skipped instead of causing an
error.
