# Update Windows agent Python runtime to 3.13.13

The Python runtime bundled with the Windows Checkmk agent has been updated
to 3.13.13. This update was motivated by CVE-2025-15467,
a stack-based buffer overflow in OpenSSL affecting CMS/PKCS#7
AuthEnvelopedData message parsing. The vulnerability does not directly affect
Checkmk, but keeping the bundled runtime current reduces exposure to
underlying library issues.
