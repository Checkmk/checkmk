# Update Windows agent Python runtime to 3.13.14

The Python runtime bundled with the Windows Checkmk agent has been updated
to 3.13.14. This update was motivated by CVE-2026-45186, a vulnerability in
libexpat (fixed in libexpat 2.8.1), and by the bundled OpenSSL being updated
to 3.0.20. Neither issue directly affects Checkmk, but keeping the bundled
runtime current reduces exposure to underlying library issues.
