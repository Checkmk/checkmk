# Network topology: Hide menu entry without permission

key        | value
---------- | ---
date       | 2026-06-12T00:00:00+00:00
version    | 2.5.0p8
class      | fix
edition    | community
component  | multisite
level      | 1
compatible | yes

Previously, the *Network topology* entry was always listed in the
monitoring menu, regardless of the user's permissions.

Users without the *Network topology* permission would see the entry but
receive a permission error when trying to open it. The entry is now only
shown to users who actually hold that permission.
