# New "Release flags" section in Global settings

The global settings now provide a dedicated **Release flags** section.

Release flags are temporary, time-limited switches that let you opt in to new
functionality during development. Each flag is scheduled for removal in a
specific Checkmk version and must not be relied on for permanent configuration.

When no release flags are available for your version, the section stays
hidden, so on a release without release flags you will not notice any change.
The flag state is stored centrally and is automatically distributed to remote
sites in distributed monitoring setups.
