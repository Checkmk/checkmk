# Fixed clipped graph legend in dashboard widgets

edition    | community

Previously, the legend of a graph in a dashboard widget was clipped to roughly two
rows. The hidden rows could not be reached, and dragging the widget taller did not
enlarge the legend area, so in graphs with many curves most metrics stayed invisible.

The legend now uses the space below the graph and scrolls when there are more entries
than fit, and resizing the widget taller enlarges the legend accordingly. In very
short widgets, where there is no room for both a usable graph and a legend, the widget
now shows the message "Either increase the widget height or disable the graph legend."
instead of a clipped, unusable legend.
