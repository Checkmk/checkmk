# Perf-O-Meter column is empty when exporting views as CSV or JSON

version    | 2.6.0b1
class      | fix

When exporting the data of a host or service view via *Export > Export
CSV* or *Export JSON*, the values of the *Perf-O-Meter* column were left
empty.

This is a regression that was introduced after Checkmk 2.3.0. The export
now again contains the Perf-O-Meter label text (for example `910 ms` or
`122 kbit/s / 84.2 kbit/s`), matching the behavior of earlier versions.
