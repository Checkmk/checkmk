# Certificate check: STARTTLS is now possible for IMAP

key        | value
---------- | ---
date       | 2026-06-02T13:38:37.972047+00:00
version    | 3.0.0b1
class      | feature
edition    | community
component  | checks
level      | 1
compatible | yes

When using "Check certificates", you can now select "IMAP STARTTLS" as
connection type.

This allows checking the certificate of IMAP servers that require a
STARTTLS procedure to use TLS (SSL), for example on port 143.

This change has already been released with Checkmk 2.5.0b1.
