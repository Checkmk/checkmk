# Certificate check: STARTTLS is now possible for LDAP

key        | value
---------- | ---
date       | 2026-06-02T13:46:00.979870+00:00
version    | 2.5.0p6
class      | feature
edition    | community

level      | 1
compatible | yes

When using "Check certificates", you can now select "LDAP STARTTLS" as
connection type.

This allows checking the certificate of LDAP servers that require a
STARTTLS procedure to use TLS (SSL), for example on port 389.

This change has already been released with Checkmk 2.5.0b1.
