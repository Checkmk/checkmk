# Parameters for notification methods: Keep search filter when editing a parameter

---------- | ---

version    | 2.4.0p32
class      | fix
edition    | cre

compatible | yes

On the page "Setup > Events > Notifications > Parameters for notification
methods", after opening a notification method you can narrow down the listed
parameters with the in-page search.

Previously, editing, adding, cloning or deleting a parameter discarded this
filter, so the page returned to showing all parameters again.

The search filter is now preserved across these actions.
