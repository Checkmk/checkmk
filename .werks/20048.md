# Notification parameters: Widen the "Call with the following parameters" input fields

key        | value
---------- | ---
date       | 2026-05-27T09:10:44.223853+00:00
version    | 2.4.0p32
class      | fix
edition    | cre
component  | notifications
level      | 1
compatible | yes

When editing the parameters of a notification method that is configured via
"Call with the following parameters" (for example custom notification
scripts), the input fields for the individual parameters were too narrow to
show their full content.

These input fields are now wider, matching the size of the other fields on
the same page.
