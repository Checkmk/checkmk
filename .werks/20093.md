# Reverting changes now reverts metric backend config

Previously, when you configured the metric backend under
"Setup > Global Settings > Site management" and then used _Revert changes_
before activating, the configuration was not removed.
The revert appeared to run through successfully, but the configuration silently
survived.

This has been fixed.
