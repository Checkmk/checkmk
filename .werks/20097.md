# Fix "Unknown password type." when configuring AWS S3 or Azure backup targets

key        | value
---------- | ---
date       | 2026-06-05T10:25:39+00:00
version    | 3.0.0b1
class      | fix
edition    | community
component  | wato
level      | 1
compatible | yes

Opening the *Site backup targets* list showed the error message
"Unknown password type." for backup targets of type *AWS S3 Bucket* or
*Azure Blob Storage* that were added or edited on 2.5, and backups for
those targets could no longer be created, restored, or triggered.
Backup targets that were configured before the update to 2.5 and not
edited afterwards were not affected.

The backup setup now saves targets correctly again. Any targets saved
in the incorrect format are fixed during the update.
