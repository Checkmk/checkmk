# Rename the "CPU utilization for appliances" ruleset

The check parameter ruleset previously titled _CPU utilization for appliances_
has been renamed to _CPU utilization (single value)_.

The old title suggested the ruleset only applied to hardware appliances, but it
is shared by a wide range of checks, including cloud resources such as Microsoft
Azure virtual machines and databases. The new title reflects that the ruleset
configures a single set of levels on one overall CPU utilization value,
independent of the underlying device type.

This is a purely cosmetic change. The ruleset's internal name
(`cpu_utilization`), its parameters and any existing rules are unaffected.
The check manuals of the checks that referenced the rule by its old title have
been updated to the new name.
