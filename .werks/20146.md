# Azure dashboards: navigable drill-down breadcrumb

The Azure dashboards now render a breadcrumb that reflects the Azure
resource hierarchy, mirroring the drill-down navigation already offered by
the Kubernetes dashboards:

    Azure (overview)
     |
     +-- Subscription
          |
          +-- Resource group
          |
          +-- Resource group

Each breadcrumb item links to the corresponding dashboard, so you can move
up and down the Azure overview -> subscription -> resource group hierarchy
with a single click instead of adjusting dashboard filters by hand.
