# Cloud edition: parent scan is no longer available

key        | value
---------- | ---
date       | 2026-06-29T11:48:00.143673+00:00
version    | 2.5.0p9
class      | fix
edition    | cloud
component  | wato
level      | 2
compatible | no

In the Cloud edition, the network parent scan is no longer available. This removes:

- the *Parent scan* action in Setup
- the parent scan background job
- the related REST API endpoint (the `parent_scan` domain-type `start` action)

Parent scan discovers a host's network parents by running `traceroute` from the Checkmk
site and turning the gateways it finds into parent hosts. This site-side traceroute is
not available in the Cloud edition, so parent scan cannot work there and has been
removed.
