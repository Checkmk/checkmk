The file Licenses.csv holds a list of all third party packages and dependencies included in Checkmk
with reference to their respective licensing.
- omd/Licenses.csv (Checkmk-Git)
- /omd/versions/default/share/doc/Licenses.csv (Checkmk installation)

The file License_texts.pdf holds all the license texts relevant for Checkmk.
- omd/License_texts.pdf (Checkmk-Git)
- /omd/versions/default/share/doc/License_texts.pdf (Checkmk installation)

For updating these files please have a look at the respective scripts update_licenses.py and
build_license_texts_pdf.py under omd/license_sources/ within the Checkmk-Git.
